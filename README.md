# 👨‍💻 Personal Portfolio - Anuj Yadav

A modern, responsive personal portfolio website built with React.js and styled-components. It showcases my skills, projects, experience, and contact information with a clean UI and smooth user experience.

---

## 🌐 Live Demo

👉 Visit My Portfolio:  
https://personal-portfolio-pth0t3hiw-anuj-kumars-projects-d34b9b68.vercel.app

---

## ✨ Features

- ⚡ Fully responsive design (mobile + desktop)
- 🎨 Modern UI/UX design
- 🌙 Dark / Light theme toggle
- 📱 Mobile-friendly navigation menu
- 📩 Contact form using Formspree
- 🔗 Social media integration (GitHub, LinkedIn, WhatsApp, Telegram, Instagram)
- 🚀 Fast performance with React optimization
- 🧠 Clean and reusable components

---

## 🛠️ Tech Stack

- React.js
- TypeScript
- Styled-Components
- React Router
- React Hash Link
- Formspree (Contact Form)
- React Toastify
- Google reCAPTCHA
